Getting Started

Quick Start (Docker)

git clone https://github.com/analytiq-hub/doc-router.git
cd doc-router
# Set up .env based on .env.example.mongodb
docker compose --profile with-mongodb up

This will start the application with MongoDB included. Access it at http://localhost:3000

Development Setup

# Set up packages/anaytiq_data/.env and frontend/.env_local
# based on example files in those folders
cd frontend; npm install; cd ..
make dev

This sets up the full development environment with hot reloading.
