Smart Document Router

An Open Source Document Understanding and Data Extraction Tool

About

The Smart Document Router is like LLamaCloud, but designed specifically for Enterprises using ERP systems. It processes unstructured documents from various sources and uses AI to automate document workflows.

Key Features

  • Ingests unstructured docs from faxes, email, and ERPs
  • Autonomous processing with LLMs and NLP
  • Human-in-the-loop design for financial accuracy
  • Direct ERP integration capabilities

Getting Started

Quick Start

git clone https://github.com/analytiq-hub/doc-router.git
cd doc-router
docker compose --profile with-mongodb up

Development Setup

npm install
npm run dev
# In another terminal:
cd backend/fastapi
uvicorn main:app --host 0.0.0.0 --port 8000
